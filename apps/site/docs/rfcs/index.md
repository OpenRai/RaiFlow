# RFCs

RaiFlow uses lightweight RFCs to avoid re-litigating the same design questions in chat.

## Planned RFCs

- [`0001`](./0001-project-framing) — Project framing
- [`0002`](./0002-observe-mode) — Observe-mode architecture
- [`0003`](./0003-event-model) — Event model and delivery semantics

## Purpose

RFCs exist to:
- define terms
- freeze important decisions
- record alternatives considered
- explain why a design was chosen
